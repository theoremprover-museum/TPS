TPS and ETPS Manuals

TPS and ETPS come with a set of manuals, which are essential for learning how to use the system effectively. (The Tables of Contents are at the ends of some manuals.)
The ETPS User's Manual, a guide to making proofs interactively (postscript version, GZIPped postscript version, PDF version)
The GRADER manual, which shows teachers how to use the Grader program to record and process grades, including records of proofs automatically graded by ETPS. (GZIPped postscript version, PDF version)
The TPS Users manual, a guide to making proofs automatically. (GZIPped postscript version, PDF version)
The TPS Short Facilities Guide, a reference guide to the most commonly used parts of the system. (GZIPped version, PDF version)
The TPS Facilities Guide, a reference guide to the whole system. (GZIPped version, PDF version)
The TPS3 Programmer's Guide, some help for programmers. (GZIPped version, PDF version)
Online (HTML) documentation for TPS and ETPS.
