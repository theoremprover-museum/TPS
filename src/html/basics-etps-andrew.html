Basics of using ETPS on Andrew

Add the following line to your .cshrc file in your Andrew home directory:

source /afs/andrew.cmu.edu/mcs/math/etps/etps-no-xset.cshrc

For more details, see "Using ETPS on Andrew" (postscript version), (PDF version).

The ETPS User's Manual (postscript version, GZIPped postscript version, PDF version) is essential for learning how to use the system effectively.
